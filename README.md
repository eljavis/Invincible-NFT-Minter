# 🦸‍♂️ Invincible NFT Minter

Welcome to the **Invincible NFT Minter**. This project is a tribute to the legendary Image Comics series, bringing the most iconic characters to the blockchain. Built with security and efficiency in mind, this smart contract allows fans to claim their favorite heroes (and villains) as immutable digital collectibles.

## 🚀 Deployment Details

The contract is live and ready for interaction on the **Arbitrum** network. By choosing Arbitrum, we ensure fast transactions and minimal gas fees—because even Omni-Man doesn't like wasting resources.

* **Contract Address:** `0x686Fc7Ab61Ffd7A79d4a07631A63d20AeC0b51Ba`
* **Network:** Arbitrum
* **Mint Price:** **0.00002 ETH**

## 🛠 How to Interact

1. **Connect:** Open your wallet (MetaMask, Rabby, etc.) and switch to the Arbitrum network.
2. **Verify:** View the contract on Arbiscan using the address provided above.
3. **Mint:** Call the `mint` function (or equivalent) and send exactly **0.00002 ETH**.
